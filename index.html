Build-A-Car

Class: Special-Topics: Cross-Platform Programming (Summer)

Skills Learned: Flutter, Dart, Cross-programming, REST APIs, Firebase Authorization, GIT

Soft Skills: Team work, Presentation Skills, Initiative, Problem-Solving, Time Management

Team size: 2

Description:
The project consisted in learning how to use the open-sourced framework Flutter. This framework allows us to create a multi-platform application from a single codebase.

In this simple application, we created an app that would show recommended cars in local dealerships (Figures 1 - 4), and the User would be able to "build" the car they want. The app has a feature where the user selects preferred their options (Figures 11 - 15), then we would show a list of cars as a results. (Figures 16 & 18) This list would be filtered from most matches to least matches and show car options available at local dealerships.

The options are: Year, Make, Model, Number of Doors, Drive, and Fuel-type.

The user can save the cars they like to a list, or to their favorites. (Figures 18)Moreover, When they select a car, they can view the map of where the car is available, the phone number of the dealership, and more details about the car.

El Paso Tourism Database

Class: Database Management

Skills Learned: MySQL, HTML, GIT, PHP, Database Management

Soft Skills: Team work, Presentation Skills, Problem-Solving, Time Management, Communication

Team size: 4

Description:

Prompt Given: El Paso Council wants a database server where they can see what tourist business had been approved to function. The database would contain Business information, Owner and Employee Information of the Business, who had approved/disapproved the license, license information, etc.

We were to create a ER-Diagram to know the relations, know what the system shall do, and create the database and the website. The website had to also allow for admin users to add, delete, or modify information while general users could only view information.
